¡¡HOLAA!! Me llamo Mario y esto es mi trabajo parcial del segundo trimestre, cuenta , si no me equivoco, un 40% de la nota final del trimestre. El 60% restante será del examen que haremos en las próximas semanas. Este trabajo integrador toma conocimientos de tres asignaturas diferentes las cuales son Lenguaje de Marcas, donde vemos como manejar paginas web como esta y se dan cosas de html, CSS y JavaScript junto con Jquery; también tenemos Sistemas Informáticos donde conocemos mejor partes del hardware de nuestro entorno como por ejemplo piezas tales como discos duros y sus tipos, placas base, procesadores, tarjetas RAM ... Por último, el tutor nos da Entornos de Desarrollo donde vemos diferentes tipos de IDEs y lo básico de algunos programas para el desarrollo de software.

En este curso tenemos 3 asignaturas más, que son Base de Datos donde damos diferentes tipos de administradores y el lenguaje predefinido para ellos, tenemos tambien Programación donde vemos un lenguaje de programación en profundidad, en este año estamos dando Java y cosas como orientación a objetos o matrices de arrays entre otras cosas, y por último tenemos FOL (formación y orientación laboral) donde damos por ejemplo como hacer un curriculum o como afrontar una entrevista de trabajo además de otras muchas cosas útiles en la vida laboral que nos espera.
